## 전체 순서

```
1. ✅ 비상주 사무실 임대차계약        ─ 완료
2. ✅ 사업자등록 신청 (프롭넷)        ─ 신청 완료, 수령 대기
   ↓
3. 🔲 사업자등록증 수령               ─ 1~3일
   ↓
4. 🔲 사업용 통장 개설                ─ 사업자등록증 필요
   ↓
5. 🔲 토스페이먼츠 가입               ─ 3~7일 심사
   ↓
6. 🔲 구매안전서비스 이용확인증 발급
   ↓
7. 🔲 통신판매업 신고 (정부24)        ─ 1~3일
   ↓
8. 🔲 Google Play 개발자 계정 등록    ─ 1~2일
   ↓
9. 🔲 앱 출시 심사 제출               ─ 수 시간~며칠
```

## 2. 토스페이먼츠 가입

- [ ] 토스페이먼츠 가입 신청
  - https://onboarding.tosspayments.com/business-registration/intro
- [ ] 필요 서류 제출
  - 사업자등록증 (프롭넷)
  - 대표자 신분증
  - 통장 사본 (정산 계좌)
- [ ] 심사 대기 (3~7일)
- [ ] 테스트 키 발급 확인
- [ ] 테스트 환경에서 결제 플로우 검증
  - 일회성 결제 (시간팩)
  - 구독 결제 (billingKey 발급 + 자동결제)
  - 환불 처리
- [ ] 라이브 키 발급
- [ ] 서버 환경변수 교체
  - `TOSS_CLIENT_KEY` → 라이브 클라이언트 키
  - `TOSS_SECRET_KEY` → 라이브 시크릿 키
  - `TOSS_WEBHOOK_SECRET` → 라이브 웹훅 시크릿
- [ ] 구매안전서비스 이용확인증 발급 (통신판매업 신고용)

## 5. Google Play 출시

### 개발자 계정

- [ ] Google Play Console 계정 생성
  - https://play.google.com/console
- [ ] 계정 유형 선택
  - **조직 계정** 추천 (상호: 프롭넷)
  - 조직 계정은 D-U-N-S 번호 필요할 수 있음
- [ ] 등록비 결제 ($25, 약 36,000원, 1회성)
- [ ] 신원 확인 절차 완료

### 앱 출시 준비

- [ ] 앱 기본 정보 입력
  - 앱 이름: Proptalk
  - 개발자: 프롭넷
  - 간단한 설명 (80자 이내)
  - 자세한 설명 (4000자 이내)
- [ ] 그래픽 자산 준비
  - 앱 아이콘 (512x512 PNG)
  - 스크린샷 (최소 2장, 폰 + 태블릿)
  - 기능 그래픽 (1024x500)
- [ ] 콘텐츠 등급 설문 작성
- [ ] 타겟 연령층 설정 (18세 이상 추천 — 비즈니스 도구)
- [ ] 데이터 보안 양식 작성
  - 수집 데이터: 이름, 이메일, 음성 파일, 전화번호
  - 제3자 공유: OpenAI (STT), Anthropic (요약), 토스페이먼츠 (결제)
  - 데이터 암호화: 전송 시 암호화 (HTTPS)
  - 데이터 삭제 요청 가능 여부: 가능 (회원 탈퇴)
- [ ] 개인정보처리방침 URL 입력
- [ ] Release APK/AAB 빌드 및 업로드
  - `flutter build appbundle --release`
- [ ] 내부 테스트 → 비공개 테스트 → 프로덕션 순서로 출시
- [ ] 심사 통과 확인

### Google Play 정책 주의사항

- [x] D방식 웹결제 (앱 내 외부결제 링크 제거) — 이미 구현
- [ ] MANAGE_EXTERNAL_STORAGE 권한 사용 사유 설명 필요
  - 파일 관리자 앱이 아닌 경우 거절될 수 있음
  - 대안: SAF(Storage Access Framework) 사용 고려
- [ ] 녹음 권한(RECORD_AUDIO) 사용 사유 설명

## 6. 추가 고려사항

### 부가통신사업 신고

- [ ] 자본금 1억 원 초과 시 과학기술정보통신부에 신고 필요
  - 1억 이하면 불필요
  - 대부분 초기 스타트업은 해당 없음

### 세금 관련

- [ ] 부가가치세 신고 (분기별/반기별)
  - 개인사업자 간이과세자: 연 매출 1억 400만원 미만 시 적용 가능
  - 일반과세자: 매출의 10% VAT 납부
- [ ] 종합소득세 신고 (매년 5월)
- [ ] 세금계산서 발행 체계 구축 (B2B 고객 대응)
- [ ] 세무사 상담 추천 (월 10~15만원)

### 보험

- [ ] 4대 보험 가입 (직원 채용 시)
- [ ] 배상책임보험 검토 (데이터 유출 대비)
